## CNC Model
This folder contains the SolidWorks files of the CNC built by us for ITSP-2018.

### Files
- Base Plate
- NEMA 17 Stepper Motor
- Rail
- Rail Ends
- Stand
- Timing Pulley
- Tray
- Electromagnet Holder

- The complete assembly as
  - SolidWorks Assembly (.SLDASM)
  - JPEG File
  - 3D PDF
